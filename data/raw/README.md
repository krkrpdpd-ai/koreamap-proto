# Raw GIS Data Staging

Place original GIS downloads here when they are available.

Recommended candidates:

- NGII/국토정보플랫폼: 대한민국 전도, 수치지형도, DEM.
- SGIS: 행정구역 경계 and coordinate conversion API results.
- 공공데이터포털: 한국도로공사 도로선형 SHP, 국토교통부 도로대장 SHP.
- VWorld: 2D Data API layers for administrative boundaries, road data, terrain data.

The current app uses `src/data/korea-map-data.js`, a lightweight game-oriented dataset built from approximate WGS84 coordinates.
